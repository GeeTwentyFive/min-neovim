# Dependencies
`git`

# Installation
Place `init.lua` into Neovim config folder
- "*%LOCALAPPDATA%\nvim*" on Windows
- "*~/.config/nvim*" on Linux & Mac

# Usage
`Tab`  - Open directory browser

`-`    - Vertical split

**-- LSP --**

`F1`   - Hover

`F2`   - Go to declaration

`F3`   - Go to definition

`F4`   - Show references
